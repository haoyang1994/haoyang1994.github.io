Redis基础介绍

发表于 2018-07-05

概述

Redis是一个开源的，基于内存的结构化数据存储媒介，可以作为数据库、缓存服务或消息服务使用。
Redis支持多种数据结构，包括字符串、哈希表、列表、集合、有序集合、位图、Hyperloglogs等。常用的是前五种。
Redis具备LRU淘汰、事务实现、以及不同级别的硬盘持久化能力，并且支持副本集和Redis Sentinel实现的高可用方案，同时还支持通过Redis Cluster实现的数据自动分片能力。
Redis的主要功能都基于单线程模型实现，也就是说Redis使用一个线程来服务所有客户端的请求；同时Redis采用了非阻塞IO，并精细地优化了各种命令的时间复杂度，这意味着：

  1. Redis是线程安全的（因为只有一个线程），其所有操作都是原子的，不会因并发产生数据异常。
  2. Redis的速度非常快
  3. 使用高耗时的Redis命令是很危险的，会占用唯一一个线程的大量处理时间，导致所有的请求都被拖慢。
